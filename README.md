# EFUI
#### aka Empathetech Flutter UI aka empathetech_flutter_ui

### TL;DR;

EFUI is a library/package for fast, high quality UI development.

"High quality" in this case == 
 - user accessible
 - user customizable
 - dynamic design to match different OS types
